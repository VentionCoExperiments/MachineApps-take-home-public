# MachineApps Take-Home Exercises

## Getting Started

1. Fork this repository
2. Navigate to your assigned exercise in `exercises/`
3. Follow the exercise-specific README

## Prerequisites

- Python 3.10+
- Node.js 16+

## Submission

1. Push your changes to your fork
2. Document your assumptions and design decisions in the exercise README
3. Share the link to your repository

Questions? Reach out to your contact.
