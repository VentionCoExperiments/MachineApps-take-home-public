## Getting Started

### Prerequisites

Before starting any exercise, ensure you have:
- **Python 3.10+** installed
- **Node.js 16+** and npm installed

### Fork This Repository

1. Click the "Fork" button in the top-right corner of this page
2. Clone your forked repository:
```bash
git clone https://github.com/YOUR_USERNAME/MachineApps-take-home.git
cd MachineApps-take-home
```

### Choose an Exercise

Navigate to the exercise directory you've been assigned:
```bash
cd exercises/example-fullstack-app
```

Read the exercise-specific README for detailed instructions.

## Completing an Exercise

1. **Read the requirements carefully** in the exercise README
2. **Set up the backend** following the backend README
3. **Set up the frontend** following the frontend README
4. **Implement the required features**
5. **Write tests** for your implementation
6. **Document your work** including any assumptions and design decisions

## Testing Your Work

Before submitting, ensure:
- [ ] All backend tests pass: `cd backend && pytest`
- [ ] All frontend tests pass: `cd frontend && npm test`
- [ ] The application runs without errors
- [ ] You've documented any setup steps or environment variables needed

## Submission

1. Ensure all your changes are committed and pushed to your fork
2. Update the exercise README with:
   - How to run your solution
   - Any assumptions you made
   - Design decisions and trade-offs
   - Time spent (optional)
3. Share the link to your forked repository

## Tips

- **Start early** - give yourself enough time to complete the exercise
- **Ask questions** - if requirements are unclear, reach out for clarification
- **Focus on quality** - we value well-tested, clean code over feature quantity
- **Document your decisions** - explain your thought process
- **Keep it simple** - don't over-engineer the solution

## Questions?

If you have any questions about the exercise or run into technical issues, please reach out to your contact.
